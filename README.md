# valiantlytix

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production (production only)

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/) (optional)

```sh
npm run lint
```
