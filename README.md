# valiantlytix

# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 16.0 or higher

# Getting started
- Clone the repository
```
git clone  <git lab template url> <project_name>
```
- Install dependencies
```
cd <project_name>
npm install
```
- Build and run the project
```
npm run dev
```
Navigate to `http://127.0.0.1:5173`
